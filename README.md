# Weather App 🌦️

A simple weather application built using React, TypeScript and Vite.

## Features
- Search weather by city
- Shows temperature and weather conditions
- Responsive UI

## Technologies Used
- React
- TypeScript
- Vite
- TailwindCSS

## Installation

Clone the repository:

git clone https://github.com/sukanya-dwaraka/weatherApp-Application.git

Install dependencies:

npm install

Run the project:

npm run dev
